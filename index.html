URL Shortener Microservice

This app fulfills the following user stories:

  • I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
  • If I pass an invalid URL that doesn't follow the valid http://www.example.com format, the JSON response will contain an error instead
  • When I visit that shortened URL, it will redirect me to my original link

Example input:

https://dandrust-url-shortener.herokuapp.com/https://freecodecamp.com

https://dandrust-url-shortener.herokuapp.com/SxeMyc


Example output:

{"original-url":"http://www.freecodecamp.com","short-url":"https://dandrust-url-shortener.herokuapp.com/SxeMyc"}



Coded by Dan Drust for completion of Free Code Camp's URL Shortener Microservice code challenge, August 2016
